# graphql-playground

GraphQL IDE可用于更好的开发工作流（GraphSQL订阅、交互式文档和协作）。

主要采用react + 自定义依赖graphql-playground-react2 实现。

## demo

使用实现效果示例

### 安装

```
yarn install
或
npm install
```

### 本地启动

```
npm start
```

### 打包

```
npm run build
```

## graphql-playground-react2

自定义发布的依赖包，以graphql-playground-react版本1.7.28为基础。
